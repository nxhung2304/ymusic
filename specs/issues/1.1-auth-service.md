## Description
Tạo `AuthService` xử lý toàn bộ logic xác thực người dùng thông qua Google Sign In + Firebase Auth.

- Implement `signInWithGoogle()` — khởi chạy Google Sign In flow, lấy credential, đăng nhập Firebase
- Implement `signOut()` — đăng xuất cả Google và Firebase
- Expose `currentUser` — trả về `User?` từ `FirebaseAuth.instance.currentUser`
- Expose `authStateChanges` — Stream `User?` để reactive listen auth state
- Wrap toàn bộ trong try-catch, throw typed exception khi fail
