## Notes

- `videoId` = YouTube video ID (ví dụ: `dQw4w9WgXcQ`) — dùng làm document ID trong Firestore để dedup tự động
- `artist` map từ YouTube channel name (có thể là `channelTitle` từ API response)
- `thumbnailUrl` ưu tiên `maxres` → `high` → `medium` fallback
- `duration` từ `youtube_explode_dart` trả về `Duration` trực tiếp — không cần parse ISO 8601 thủ công
- Model này được tái dùng ở: Phase 3.2 (search), Phase 3.4 (Isar cache), Phase 5.1 (History), Phase 6.1 (LikedSong), Phase 7.1 (Playlist)
