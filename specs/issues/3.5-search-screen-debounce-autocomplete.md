## Implementation Checklist

### 1. State Management — `SearchNotifier`

Tạo `lib/features/search/presentation/providers/search_notifier.dart`:

```dart
@freezed
class SearchState with _$SearchState {
  const factory SearchState.initial() = _Initial;
  const factory SearchState.loading() = _Loading;
  const factory SearchState.suggestions(List<SongModel> items) = _Suggestions;
  const factory SearchState.results(List<SongModel> items) = _Results;
  const factory SearchState.empty(String query) = _Empty;
  const factory SearchState.error(String message) = _Error;
}
```

`SearchNotifier extends AutoDisposeAsyncNotifier<SearchState>`:
- `search(String query)`: gọi `YouTubeDatasource.search(query)` → emit `results` hoặc `empty`
- `suggest(String query)`: tương tự nhưng emit `suggestions` (dùng chung `YouTubeDatasource.search`)
- `clear()`: emit `initial`
- Debounce 500ms implement bằng `Timer` trong notifier (cancel timer cũ khi method gọi lại)
- Throws: catch `YouTubeException` → emit `error`

Provider:
```dart
final searchNotifierProvider =
    AutoDisposeAsyncNotifierProvider<SearchNotifier, SearchState>(
  SearchNotifier.new,
);
```

### 2. SearchScreen UI — `lib/features/search/presentation/screens/search_screen.dart`

Layout:
```
Column
  ├── SearchBar (top)
  └── Expanded
        ├── [if suggestions] SuggestionDropdown (overlay hoặc ListView)
        ├── [if results] ListView.builder(SongTile)
        ├── [if loading] LoadingView
        ├── [if empty] EmptyView
        └── [if initial] InitialSearchView ("Search for music")
```

SearchBar widget:
- `TextField` với `InputDecoration` có clear button (khi có text)
- `suffixIcon`: loading spinner khi debouncing, clear icon khi có text
- `onChanged` → gọi `notifier.onQueryChanged(value)` (trigger debounce + suggest)
- `onSubmitted` → gọi `notifier.search(value)` trực tiếp (bypass debounce)

Debounce trong notifier (KHÔNG dùng external debounce package):
```dart
Timer? _debounceTimer;

void onQueryChanged(String query) {
  _debounceTimer?.cancel();

  if (query.isEmpty) {
    clear();
    return;
  }

  // Show loading immediately
  state = const AsyncData(SearchState.loading());

  _debounceTimer = Timer(const Duration(milliseconds: 500), () {
    suggest(query);
  });
}
```

## Notes

- **Debounce không dùng package bên ngoài** — dùng `dart:async` `Timer` thuần để giữ dependency tối thiểu
- `suggest` và `search` dùng chung `YouTubeDatasource.search()` — không tạo 2 endpoint riêng; chỉ khác nhau ở `SearchState` emit ra
- `AutoDispose` provider: khi rời khỏi SearchScreen, debounce timer được cancel tự động khi provider dispose
- `SongTile.onTap` → SnackBar là placeholder; Phase 3.9 sẽ replace bằng `AudioPlayerService.play(song)`
- Không cache search results ở task này — Phase 3.10 sẽ thêm Isar cache layer
- Suggestion dùng kết quả real từ YouTube (không hardcode) — UI/UX sẽ phụ thuộc vào latency thực tế
