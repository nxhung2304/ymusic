## Description
Cấu hình `ThemeData` dark mode làm theme mặc định và wire vào `MaterialApp` trong `app.dart`.

- Tạo `lib/core/theme/app_theme.dart` với `ThemeData` dark mode đầy đủ

- Setup `lib/main.dart` với `ProviderScope` (Riverpod) wrapping app
- Dark mode là default, không hỗ trợ light mode
