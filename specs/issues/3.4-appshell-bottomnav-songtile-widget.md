### 2. Tạo SongTile widget (`lib/features/search/presentation/widgets/song_tile.dart`)

```dart
class SongTile extends StatelessWidget {
  final Song song;
  final VoidCallback? onTap;
  final VoidCallback? onMoreTap;

  const SongTile({
    required this.song,
    this.onTap,
    this.onMoreTap,
    super.key,
  });
}
```

Layout (xem `home.html`):
- `ListTile` hoặc custom `Row`:
  - **Leading:** `CachedNetworkImage` thumbnail, kích thước `48×48`, `borderRadius: 4px`, `boxFit: cover`
    - `placeholder`: `Container` màu `AppColors.surface` với icon nhạc
    - `errorWidget`: icon nhạc fallback
  - **Title:** song title, `AppTypography.body` màu white, max 1 line + overflow ellipsis
  - **Subtitle:** artist name, `AppTypography.caption` màu `AppColors.textSecondary`, max 1 line + overflow ellipsis
  - **Trailing:** `Row` gồm duration text (`mm:ss` format) + `IconButton` với `Icons.more_vert`
- `onTap` callback trên toàn tile
- `onMoreTap` callback trên nút "more_vert"

Duration format helper (private):
```dart
String _formatDuration(Duration duration) {
  final minutes = duration.inMinutes;
  final seconds = duration.inSeconds % 60;
  return '${minutes.toString().padLeft(2, '0')}:${seconds.toString().padLeft(2, '0')}';
}
```
