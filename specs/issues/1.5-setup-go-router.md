## Description
Cấu hình `go_router` cho toàn bộ app với shell route, protected routes, và redirect logic.

Route map theo CLAUDE.md:
- `/` → SplashScreen (redirect sau khi check auth)
- `/login` → LoginScreen (public)
- `/home` → AppShell (ShellRoute — BottomNav persistent)
  - `/home` → HomeScreen
  - `/search` → SearchScreen
  - `/library` → LibraryScreen
- `/player` → FullPlayerScreen (push over shell)
- `/video/:videoId` → VideoPlayerScreen
- `/playlist/:playlistId` → PlaylistDetailScreen
- `/settings` → SettingsScreen

Redirect logic:
- Nếu `authStateProvider` là `null` và route cần auth → redirect `/login`
- Nếu đã auth và đang ở `/login` hoặc `/` → redirect `/home`
