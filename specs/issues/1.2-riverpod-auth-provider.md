## Description
Expose `AuthDatasource` qua Riverpod providers dùng `@riverpod` codegen để toàn bộ app có thể reactive theo auth state.

- `authDatasourceProvider` — dùng `@riverpod` function provider, trả về `AuthDatasource` singleton
- `authStateProvider` — dùng `@riverpod` StreamNotifier hoặc function provider, trả về `Stream<User?>`
- `currentUserProvider` — dùng `@riverpod` function provider, derived từ `authStateProvider`
- Đặt file trong `lib/features/auth/presentation/providers/auth_provider.dart`

```dart
@riverpod
AuthDatasource authDatasource(AuthDatasourceRef ref) => AuthDatasource();

@riverpod
Stream<User?> authState(AuthStateRef ref) =>
    ref.watch(authDatasourceProvider).authStateChanges;

@riverpod
User? currentUser(CurrentUserRef ref) =>
    ref.watch(authStateProvider).value;
```
