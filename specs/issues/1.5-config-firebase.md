## Description
Tích hợp Firebase vào Flutter project iOS — khởi tạo Firebase App, cấu hình Auth và Firestore.

- Tạo Firebase project trên Firebase Console
- Đăng ký iOS app vào Firebase project
- Download và thêm `GoogleService-Info.plist` vào `ios/Runner/`
- Chạy `flutterfire configure` để generate `firebase_options.dart`
- Khởi tạo Firebase trong `main.dart` trước khi app chạy
- Cấu hình Firestore Security Rules cơ bản

## Implementation Checklist
- [ ] Tạo Firebase project tại console.firebase.google.com
- [ ] Enable Authentication (Google Sign-In provider)
- [ ] Enable Firestore Database (production mode)
- [ ] Đăng ký iOS app với bundle ID (ví dụ: `com.ymusic.dev`)
- [ ] Download `GoogleService-Info.plist` → đặt vào `ios/Runner/`
- [ ] Thêm vào Xcode project nếu cần
- [ ] Cài `flutterfire_cli`: `dart pub global activate flutterfire_cli`
- [ ] Chạy: `flutterfire configure` → chọn iOS only → generate `lib/firebase_options.dart`
- [ ] Update `main.dart`:
  ```dart
  await Firebase.initializeApp(options: DefaultFirebaseOptions.currentPlatform);
  ```
- [ ] Deploy Firestore Security Rules:
  ```
  rules_version = '2';
  service cloud.firestore {
    match /databases/{database}/documents {
      match /users/{userId}/{document=**} {
        allow read, write: if request.auth != null && request.auth.uid == userId;
      }
    }
  }
  ```
- [ ] Test Firebase connection (không crash khi khởi động)
- [ ] `flutter analyze` — 0 warnings
