## Description
Tạo các file constants tập trung để quản lý màu sắc, API endpoints, và strings dùng xuyên suốt app.

- Tạo `lib/core/constants/app_colors.dart` — định nghĩa color palette dark theme
- Tạo `lib/core/constants/app_endpoints.dart` — YouTube Data API v3 base URL, lrclib.net endpoint

- Tạo `lib/core/constants/app_constants.dart` — misc constants (timeouts, TTL, API key placeholder)
